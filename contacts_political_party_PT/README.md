# Módulo de Campos Personalizados para Contactos

## Funcionalidades
- Adiciona campos de filiação, dados pessoais, partidários e eleitorais.
- Permite gerir setoriais de atuação a partir do menu Configurações.
- Valida CNF (9 dígitos), Título Eleitoral (12 dígitos) e regras de desfiliação.

## Instalação
1. Copie a pasta `meu_modulo_contatos` para o diretório `addons`.
2. Actualize a lista de módulos em Aplicações.
3. Instale o módulo "Campos Personalizados para Contatos".

## Utilização
- Os campos aparecem nos formulários de contacto, apenas quando a opção "É filiado?" está activa.
- Os setoriais podem ser geridos em: Configurações → Contactos → Setoriais de Atuação.
